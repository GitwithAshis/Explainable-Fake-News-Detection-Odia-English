# Explainable Fake News Detection for Odia and English

## 📖 Overview

This repository contains the implementation of my M.Tech research project:

**Explainable AI-based Fake News Detection and Source Credibility Analysis for Odia and English News using the XLM-RoBERTa Transformer Model.**

The project uses transformer-based Natural Language Processing (NLP) and Explainable AI (XAI) to classify news as fake or real while providing model interpretability through Captum.

## ✨ Features

- Fake News Detection
- Odia and English Language Support
- XLM-RoBERTa Transformer Model
- Explainable AI using Captum
- Source Credibility Analysis
- Confidence Score Prediction

## 🛠 Technologies

- Python
- PyTorch
- Hugging Face Transformers
- Captum
